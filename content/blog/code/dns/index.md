一般常用配置：

- 国内 DNS： `223.5.5.5` `119.29.29.29` `114.114.114.114`
- 远程 DNS：`8.8.8.8` `1.1.1.1` 
- Bootstrap：`宽带下发的默认 DNS`

有条件尽量使用 DoH 或者 DoT。
